# QR Code Generator

A simple Python project that generates QR codes from text or URLs and saves them as PNG images. This project is built using the **PyQRCode** and **PyPNG** libraries.

## Features

- Generate QR codes from any text or URL
- Save QR codes as PNG images
- Simple and beginner-friendly Python implementation
- Lightweight and easy to use

## Technologies Used

- Python
- PyQRCode
- PyPNG

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Divyasrinivas09/QR-Code-Generator.git
```

2. Navigate to the project directory:

```bash
cd QR-Code-Generator
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the Python script:

```bash
python qr_code_generator.py
```

Enter any text or URL when prompted. The generated QR code will be saved as:

```
QR_Code.png
```

## Example

**Input**

```
https://www.github.com
```

**Output**

A PNG image containing the corresponding QR code.
